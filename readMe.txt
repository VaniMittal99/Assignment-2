# Library Management System

## Description
The Library Management System project is designed to efficiently manage and organize the operations of a library. It allows librarians to catalog books, manage checkouts and returns, keep track of patrons, and more.

## Building and Running the Project
To build and run the Library Management System, follow these steps:

1. **Clone the Repository:** 
git clone https://github.com/vanimittal99/library-management-system.git

2. **Navigate to the Project Directory:**
cd library-management-system

3. **Compile the Code:**

To compile the code and generate the debug and optimized binaries, run the following command:
make

4. **Run the Debug Version:**
./debug_program

5. **Run the Optimized Version:**
./optimized_program

6. **Clean Up:**
To remove the generated binaries, run:
make clean
